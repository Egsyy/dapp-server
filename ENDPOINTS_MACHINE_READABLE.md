# Machine-readable list of endpoints

This file contains a machine-friendly listing of discovered endpoints in three formats: JSON, CSV, and a Markdown table. The list was generated by scanning feature slices under `features/` and key infrastructure files. `route` values are the final mounted routes (each feature group is mounted under `/api/` by the service discovery extensions; hubs are mounted under `/hub/`).

Note: `auth` is "explicit" when the code adds `.RequireAuthorization()` to the route; otherwise it is marked `unknown` (enforced by global middleware or attribute usage).
